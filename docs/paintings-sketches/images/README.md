# Visual Gallery Images

Drop your images here (.jpg, .jpeg, .png, .webp).

The images will be automatically loaded and displayed in the horizontal scrollable gallery on the visual/paintings-sketches page.

**Supported formats:**
- jpg / jpeg
- png
- webp

**Recommended:**
- Keep image dimensions consistent (e.g., 800x600 or similar aspect ratio)
- Optimize file sizes for web (aim for < 500KB per image)
